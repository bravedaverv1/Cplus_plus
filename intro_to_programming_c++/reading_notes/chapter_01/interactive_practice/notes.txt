1.15.1: Complete the program to print out the picture of a tree and bush.

Complete the program below, named treebush.cpp, to print out the 
child-like picture of a tree and bush below. The output of this program 
should match the following, exactly:
      o
    oooo
  ooooooo
  oooooooo
   ooIIo o       ooo
   o II         oooooo
_____II_________ oooo____

#include <iostream>

using namespace std;
                   
int main()
{
   cout << "      o" << endl;
   cout << "    oooo" << endl;
   cout << "  ooooooo" << endl;
   cout << "  oooooooo" << endl;
   cout << "   ooIIo o       ooo" << endl;
   cout << "   o II         oooooo" << endl;
   cout << "_____II_________ oooo____" << endl;
   return 0;
}
